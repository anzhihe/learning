### 子查询

一个查询的结果做为另一个查询的条件。



查询年龄最大学生的信息

```
select * from tbl_students where age = (select max(age) from tbl_students );
```

查询黑龙江省的所有市

```
select * from areas where parent_id = (select id from areas where name = '黑龙江省'); 
```
