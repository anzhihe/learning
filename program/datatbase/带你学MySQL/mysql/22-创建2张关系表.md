### 创建班级表

```
create table tbl_classes (
	id int not null primary key auto_increment,
	name varchar(45) not null 
)engine=innodb default charset=utf8;
```

### 创建学生表

```
create table tbl_students (
	id int not null primary key auto_increment,
	name varchar(50) not null,
	gender varchar(1) not null,
	age int(11) not null,
	cls_id int(11)
)engine = innodb default charset=utf8;
```
